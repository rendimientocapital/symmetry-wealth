# Symmetry Wealth API
Backend Node.js + TypeScript para Symmetry Wealth.

## Stack
- Express + TypeScript
- Supabase PostgreSQL + pgvector
- Claude API (Anthropic)
- Deploy en Railway

## Endpoints principales
- GET /health
- GET /auth/callback
- GET /market/financials
- GET /portfolio
- POST /claude/ask
- POST /webhooks/wix
